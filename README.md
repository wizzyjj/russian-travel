# Практическая работы №3 Russian-travel

### Обзор
Итогом практической работы №3 стал одностраничный сайт о путешествиях по России.
На странице реализовано адаптивное отображение страницы с 4я брейкпоинтами:
* 320px;
* 768px;
* 1024px;
* 1280px;



### Технологии
* HTML5
* CSS3
* BEM
* Flexbox
* Grid Layout
